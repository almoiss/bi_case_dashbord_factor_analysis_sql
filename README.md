# bi_case_dashbord_factor_analysis_sql
BI Case: Power BI Dashboard, Factor Analysis and SQL


## Цель проекта
Показать навыки BI-аналитики: анализ данных, визуализация показателей,
факторный анализ выручки и решение SQL-задач, типичных для аналитической работы.

## Что сделано

### Power BI
- Подготовка и объединение данных
- Расчёт ключевых показателей (выручка, прибыль)
- Интерактивный дашборд с фильтрами
- Факторный анализ влияния цены и объёма на выручку по SKU

### SQL
- Запрос для поиска сотрудников с менее чем 3 задачами
- Создание представления (VIEW) для аналитической выгрузки
- Реализация логики справочника с историчностью (SCD Type 2)

## Инструменты
- Power BI
- PostgreSQL

## Результаты
- Дашборд для анализа финансовых показателей
- Подготовлены SQL-решения для аналитических задач
